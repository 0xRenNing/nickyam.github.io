记一次手动发现自己的页面被其他站长引用

2020.02.07 虞侃 ⓶科技  热度 ℃

今天在搜索优化建站相关信息的时候，发现一个个人站长用hexo建的站点，里面有一篇文章《Hexo 框架下404页面的设置》，在最后的Reference部分引用了三篇文章，第一篇就是我的《Hexo博客添加腾讯404公益页面》，链接是https://www.nickyam.com/tech/hexo-add-tencent-404-page.html。

这是我第一次手动发现自己的页面被引用，Mark！

我也仔细阅读了他的文章，集众家之长，更加可以借鉴！故引用之，大家也可以参考！

详见《Hexo 框架下404页面的设置》。
