2021年1月1日，特斯拉中国宣布国产新款 Model 3与国产 Model Y正式上市，两款新车均有两种配置，国产新款标准续航 Model 3售价为24.99万元，国产新款 Model Y售价为33.99万元起，这也算是2021年第一天献上的一份好礼。

笔者在10月31日下单了标续版本的 Model 3，12月初通知我匹配成功。我本来也打算去提车了，一方面是因为磷酸铁锂电池的冬季表现，另一方面是因为改款的声音越来越大。

今天特斯拉的官宣，证明我的等待是值得的。我去官网看了我的订单，状态已经更新，所以就等着排队匹配好了。
