关于“出口免抵退”的实务笔记

2017.03.12 虞侃 ⓹工作  热度 ℃

在信贷审查中，如何核实企业的生产经营情况和销售收入是非常重要的命题。当企业的财务报表或多或少存在水分的时候，作为信贷人员的我们就需要从其他方面寻找突破口，比如机器设备的产能、订单、银行流水、企业水电费等等都可以作为企业经营的佐证。今天笔者整理的一个佐证，是企业完税情况。当政府征税和企业纳税完全按照法律法规进行的前提下和在“营改增”的大背景下，通过增值税完税情况和行业平均税负率大致能够推断企业的当年销售收入。而对于出口企业，因为存在出口退税环节，通过企业完税情况倒推企业销售收入就不是那么直观了。在部分制造业出口企业的信贷实务摸索后，笔者力求在信息不对称的有限企业信息下推算企业销售收入，有一点心得（本文为笔者某业务交流PPT的文字简化提纲总结版），与大家分享。

一、基本概念

1、出口退税的概念

出口退（免）税是国际贸易中通常采用的并为世界各国普遍接受的，目的在于鼓励各国出口货物公平竞争的一种退还或免征间接税的税收措施。出口退税能够避免国际贸易中间接双重征税的问题，并且不需要各国间签订避免双重征税的对等协议，是一种自主行为而形成的国际惯例。

我国的出口货物退（免）税是指在国际贸易业务中，对我国报关出口的货物退还或免征其在国内各生产环节和流转环节缴纳的增值税和消费税。开展“营改增”试点后，我国对境内单位和个人提供国际运输服务、向境外单位提供的研发服务和设计服务，使用增值税零税率，实行出口退税的范围扩大到服务出口。

2、出口退税对象的范围

根据现行政策规定，可以申请办理出口退税的企业和其他单位包括：

  • 经营出口货物劳务的出口企业（主要是日常出口企业，包括我们平时碰到的生产型和贸易型的企业）
  • 经营视同出口货物劳务的企业和其他单位（包括对外援助的出口货物企业等）
  • 提供增值税零税率应税服务并认定为增值税一般纳税人的单位和个人（提供国际运输服务、向境外单位提供的研发服务和设计服务）

3、出口退税对象的范围

  • 必须是增值税、消费税征收范围内的货物。增值税、消费税的征收范围，包括除直接向农业生产者收购的免税农产品以外的所有增值税应税货物，以及烟、酒、化妆品等11类列举征收消费税的消费品。（未征不退原则）
  • 必须是报关离境出口的货物。所谓出口包括自营出口和委托代理出口两种形式。区别货物是否报关离境出口，是确定货物是否属于退（免）税范围的主要标准之一。凡在国内销售、不报关离境的货物，除另有规定者外，不论出口企业是以外汇还是以人民币结算，也不论出口企业在财务上如何处理，均不得视为出口货物予以退税。
  • 必须是在财务上作出口销售处理的货物。出口货物只有在财务上作出口销售处理后，才能办理退（免）税。也就是说，出口退（免）税的规定只适用于贸易性的出口货物，而对非贸易性的出口货物，如捐赠的礼品、在国内个人购买并自带出境的货物（另有规定者除外）、样品、展品、邮寄品等等，因其一般在财务上不作销售处理，故按照现行规定不能退（免）税。
  • 必须是已收汇并经核销的货物。按照现行规定，出口企业申请办理退（免）税的出口货物，必须是已收外汇并经外汇管理部门核销的货物。（企业销售收入的确认是权责发生制，但是出口退税的基础可以类比收付实现制）

4、出口退税的方式

  • 免抵退税办法：生产企业出口自产货物和视同自产货物及对外提供加工修理修配劳务，以及列名生产企业出口非自产货物，免征增值税，相应的进项税额抵减应纳增值税额（不包括适用增值税即征即退、先征后退政策的应纳增值税额），未抵减完的部分予以退还。

    ![][image-1]

    应纳税额若小于零说明内销应纳增值税抵顶应退外销进项后仍有余额，分两种情况：

    • 内销的销项本身就小于内销进项造成的小于零。
    • 内销的销项减内销的进项大于零，再减外销的进项后小于零。第一种情况如果全部退税势必将内销留底税额一并退还企业。*==故有规定退税限额：外销金额退税率==**

  • 免退税办法：不具有生产能力的出口企业（外贸企业）或其他单位出口货物劳务，免征增值税，相应的进项税额予以退还。

二、出口免抵退计算方法

对于企业财会人员，按照纳税申报表的格式填写当月值就是计算增值税额的逻辑。但是结合出口退税的特点可以提炼特点归纳如下：

1、生产企业“免抵退”

　　1.当期应纳税额的计算

　　当期应纳税额=当期销项税额-（当期进项税额-当期不得免征和抵扣税额）

　　当期不得免征和抵扣税额=当期出口货物离岸价x外汇人民币折合率x（出口货物适用税率-出口货物退税率）-当期不得免征和抵扣税额抵减额

　　当期不得免征和抵扣税额抵减额=当期免税购进原材料价格x（出口货物适用税率-出口货物退税率）

　　上述当期应纳税额如为负数，则为当期期末出现留抵税额。

　　2.当期免抵退额的计算

　　当期免抵退税额=当期出口货物离岸价x外汇人民币折合率x出口货物退税率-当期免抵退税额抵减额

　　当期免抵退税额抵减额=当期免税购进原材料价格x出口货物退税率

　　3.当期应退税额的计算

　　如当期期末留抵税额≤当期免抵退税额，则当期应退出口增值税等于当期期末留抵税额

　　如当期期末留抵税额>当期免抵退税额，则当期应退出口增值税等于当期免抵退税额。

　　因此当期免抵退税额也可以理解为当期出口货物增值税退税的最高限额。

　　（具体例略）

​ 对于出口免抵退的计算，最重要的是出口免抵退税额对内销部分增值税的抵扣判断。下面是三种不同情况的图示：

b.应纳税额为负，内销应纳税额为正。

2、外贸企业“免退”

外贸企业出口货物后，免征销售环节的销项税额，依据外贸出口企业为出口购买商品取得的增值税专用发票注明的商品不含税金额，再乘以退税率办理退税，即：外贸企业出口货物增值税应退税=增值税退（免）税计税依据x出口货物退税率增值税退（免）税计税依据为购进出口货物的增值税专用发票注明的金额。

3、进料加工和来料加工

进料加工是指国内有外贸经营权的单位用外汇购买进口部分或全部原料、材料、辅料、元器件、配套件和包装物料加工成品或半成品后再返销出口国外市场的业务。

来料加工是指外商提供全部原材料、辅料、零部件、元器件、配套件和包装物料，必要时提供设备，由承接方加工单位按外商的要求进行加工装配，成品交外商销售，承接方收取工缴费，外商提供的作价设备价款，承接方用工缴费偿还的业务。

在考虑出口免抵退时，

来料加工视同免税：

  1. 来料加工出口货物免征增值税
  2. 加工企业取得的加工费收入免征增值税
  3. 出口货物说好用的国内货物所支付的进项税额不得抵扣，转入生产成本，其国内配套的原材料的已征税款也不予退税
  4. 进口时免征关税，出口时对成品也不征收出口关税

进料加工视同自产自销：

  1. 进料加工的进口料件一般是免征增值税的，出口时免税进口料件这一部分对应的进项税额是不存在的，所以我们在计算免抵退税时，要==计算一个免抵退税额不得免征和抵扣税额的抵减额==
  2. 按照加工后复出口的数量免征进口关税，出口时这部分材料免征出口关税

三、应税销售估算公式

1、“免抵退”公式基本演绎

根据增值税的基本定义

当期应纳税额

= 当期销项税额-（当期进项税额-当期不得免征和抵扣税额）-（上期留抵税额-上期应退税额）

= 当期销项税额-当期进项税额+当期不得免征和抵扣税额-（上期留抵税额-上期应退税额）

≈ 当期内销销售×销项税率 -（当期销售进项+存货增加部分）×进项税率+当期出口货物离岸价×人民币外汇牌价×（征税率-退税率）-免税购进原材料价格×（征税率－退税率）+当期进项税额×(间接出口销售额÷本月销售总额)–（上期留抵税额-上期应退税额）

≈ -出口部分进项税额 +（当期内销销售×销项税率-当期内销部分进项税额）-当期存货增加部分进项税额+（海关出口销售-免税购进原材料成本）×（征税率－退税率）+当期进项税额×(间接出口销售额÷本月销售总额)

对纺织企业，目前有如下特点：

  • 出口货物适用税率为0，征税率17%，退税率0-17%都有，但从生产流程看从纱线这道开始目前退税基本在17%,下页简化公式采用退税17%

  • 内销销项税率=进项税率=17%，故

    当期内销销售×销项税率-当期内销部分进项×进项税率≈当期内销销售×行业税负率

  • 存货增减部分在一年范围内从报表获取，如能取得存货中原材料的增减，则取原材料的变化

  • 免税购进原材料价格×（出口货物征税率－出口货物退税率）中若退税率为17%，则（出口货物征税率－出口货物退税率）为0，该项整体为0。若退税率不为17%，则需了解是否有免税购进行为。

  • 当期进项税额×(免税销售额÷本月销售总额)暂不考虑，故下页开始所指的应纳税额偏小。

税负率=应纳税额/应税销售=（销项税额-进项税额）/应税销售=（应税销售×0.17-进项成本×0.17）/应税销售=（（应税销售-应税成本）/应税销售））×0.17≈≈ 增值率×0.17

最终

当期内销销售（偏小）

≈（当期应纳税额+当期存货增加部分×17%+出口海关数据×（1-增值率）×17%）/税负率

≈ 当期应纳税额/(0.17×增值率）+存货增量/增值率+海关出口数据×（1/增值率-1）

全部应税销售 ≈ 当期应纳税额/(0.17×增值率）+存货增量/增值率+海关出口数据/增值率

故在粗略估算尺度上，应税销售与应纳税额、增值率、存货增量、海关出口数据相关。

值得注意的是，前述计算中都不包括免税销售。

2、“免抵退”销售估算（退税率不为17%）

同上可以推得

全部应税销售 ≈ （当期应纳税额+17%×存货增量+退税率×海关出口销售+（17%-退税率）×海关进口）/（17%×增值率）
