UBI项目 GoodDollar ：使用区块链对抗贫穷

2023.05.27 虞侃 ⓶科技  热度 ℃

近期， 区块链项目 GoodDollar 从 Fuse 网络扩展到了更为普及的 Celo 网络。之前笔者在 Fuse 网络的 G$ 代币一直无法 bridge 到主流网络，而现在 Celo 的参与使得这一切成为可能。该项目的现实价值得到提升。

GoodDollar是一个基于区块链技术的项目，旨在通过分发全球通用的基本收入（UBI）来解决全球贫困问题，通过创新的区块链技术和基本收入概念来推动社会公正和减少全球贫困。它提供了一种可持续且包容性的方式，让人们能够在全球范围内获得经济支持和金融包容性。具体特点如下：

  1. 基本收入概念：基本收入是指向每个人提供固定金额的资金支持，而无需履行特定的条件或工作。GoodDollar通过区块链技术实现了这一概念，向全球的参与者分发数字资产。

  2. 货币系统：GoodDollar创建了一种名为GoodDollar（G$）的数字货币，每天产生的GoodDollar供应量与全球人口数量成比例增长。这种货币系统旨在确保可持续、公平和包容性。

  3. 通用基本收入：GoodDollar项目通过每天向参与者分发GoodDollar来实现全球通用的基本收入。每个人可以注册并获得一定数量的GoodDollar，这些数字资产可以用于交易、存储或转移给其他人。

  4. 社区建设：GoodDollar鼓励社区参与和贡献，以推动项目的发展和扩展。社区成员可以通过参与学习、开发工具和参与治理等方式来支持项目的发展。

注册该项目以后，你可以获得一个0x开头的区块链钱包地址，每天在 Fuse 网络和 Celo 网络均可以领取 G$ 代币，每个网络均可以领导50个左右的代币。目前价值不高，但如果后续项目成功，可能会有意外的收获。

而通过这个项目，你也可以0成本熟悉区块链中swap、gas等概念和操作。
